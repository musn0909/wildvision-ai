# 🐾 WildVision AI

WildVision AI is a deep learning-based animal classification system built using TensorFlow and Streamlit.

## Features

- AI-powered animal recognition
- Modern Streamlit UI
- Top-3 predictions
- Confidence analysis
- Real-time image upload
- Transfer learning using MobileNetV2

## Supported Animals

- Dog
- Horse
- Elephant
- Butterfly
- Chicken
- Cat
- Cow
- Sheep
- Spider
- Squirrel

## Technologies Used

- Python
- TensorFlow
- Streamlit
- Plotly
- PIL
- NumPy

## Run Locally

```bash
streamlit run app.py
